flex-direction

Controls the main axis direction of a flex container.

  • row: left to right (default)
  • row-reverse: right to left
  • column: top to bottom
  • column-reverse: bottom to top

order

Determines the order that elements will appear in a flex container. Default order is 0.

flex-grow

Defines the ratio of space an item will take up within a flex container in relation to its siblings. For example, a flex container with three children divs with flex-grow of 1 will take up the same amount of space. Whereas, if one of those children had a flex-grow of 2, it would take up twice as much space as it's siblings.

  • flex-grow default is 0
  • can't be a negative value
